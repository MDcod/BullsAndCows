# BullsAndCows
Тестовое задание

## Решение базового и продвинутого уровней
Два режима игры:
1. Базовый уровень (пользователь угадывает число)
2. Продвинутый уровень (компьютер угадывает число)

### Генерация сочетаний 4х цифр

Все сочетания 4х цифр были сгенерированы программно. Метод для генерации находится в классе CombinationGeneration.cs

Для ускорения работы, программы берет, уже готовый, список сочетаний из текстового файла GenerationСombinationFourdigitNumbers.txt

Так же можно сделать, чтобы сочетания генерировались, каждый раз при запуске программы. Это избавит от зависимости с внешним файлом.
