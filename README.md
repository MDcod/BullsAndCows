# BullsAndCows
Тестовое задание

Папку UnitTest необходимо поместить в одну директорию с папкой BullsAndCows. То есть на уровень выше.

Базовый уровень находится на ветке master

## Программа с рещением продвинутого уровеня находится на ветке GuessComputer
Два режима игры:
1. Базовый уровень (пользователь угадывает число)
2. Продвинутый уровень (компьютер угадывает число)
